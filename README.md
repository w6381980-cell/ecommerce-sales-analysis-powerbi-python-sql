# 📊 E-Commerce Sales Analysis Dashboard

## 🚀 Project Overview
This project focuses on analyzing an E-commerce dataset to uncover key business insights such as revenue trends, customer behavior, product performance, and delivery efficiency.

The project is built using:
- SQL Server (Data Storage)
- Python (Data Cleaning & Analysis)
- Power BI (Dashboard Visualization)

---

## 🎯 Objective
To analyze sales data and identify:
- Revenue drivers
- Top-performing products
- Customer distribution
- Sales trends
- Delivery performance

---

## 🧰 Tools & Technologies
- SQL Server
- Python (Pandas, NumPy)
- Power BI
- VS Code

---

## 🧹 Data Cleaning (Python)
- Removed duplicate records
- Handled missing values
- Converted date formats
- Merged multiple datasets

---

## 📊 Key Insights
- 💰 Total Revenue: **13.6M**
- 🏙️ Top City: São Paulo
- 📅 Peak Sales Months: May, August
- 📦 Top Categories: Beauty, Watches, Home
- 🚚 Delivery: Most orders delivered earlier than expected

---

## 📈 Dashboard Features
- KPI Cards (Revenue, Orders, Customers)
- Monthly Sales Trend
- Top Cities Analysis
- Category Performance
- Interactive Filters (Month, State)

---

## 💡 Business Recommendations
- Focus marketing on top cities
- Promote high-performing categories
- Improve low-sales months with campaigns
- Highlight fast delivery as a competitive advantage
